✊ Rock Paper Scissors Game<br>

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. This project is a beginner-friendly implementation of core frontend concepts like DOM manipulation, event handling, and basic game logic.<br>

🚀 Features<br>

User vs Computer gameplay<br>
Random computer choice generation<br>
Instant result display (Win / Lose / Draw)<br>
Score tracking system<br>
Responsive and clean UI<br>

🛠️ Technologies Used<br>

HTML – Structure of the game<br>
CSS – Styling and layout<br>
JavaScript – Game logic and interactivity<br>

📂 Project Structure<br>
rock-paper-scissors/<br>
│<br>
├── index.html<br>   
├── style.css<br> 
└── script.js<br>   

🎮 How to Play<br>

Choose one option: Rock, Paper, or Scissors<br>
The computer will randomly select its choice<br>
The result will be displayed instantly:<br>
Rock beats Scissors<br>
Scissors beats Paper<br>
Paper beats Rock<br>
Scores will update based on the result<br>
